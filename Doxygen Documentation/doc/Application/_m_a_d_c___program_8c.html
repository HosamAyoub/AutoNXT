void MADC_VoidInit ( void  )

This function is responsible of Initializing the adc and it's options and modes.

Parameters
none
Returns
none

This function will deal with the different modes and options and configurations of adc like:

  1. Set the resolution of the conversion of the channels of adc
  2. Set the mode of adc define the status either in single channels or multiple channels mode and single and also define the method of conversion either snigle or continuously
  3. Set the method of storing the data in data register after the conversion
  4. Set the time to notificate the system or raising a bit flag either after each conversion or after all conversions and select if the interrupt is desired or not
  5. Set the sampling time for each channel and also set the channels of conversion in the order and set the number of channels in the system
  6. Set the dma options or disable the dma in general
  7. Select the prescaler of the peripheral
