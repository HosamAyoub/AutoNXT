u32 HLDR_u32MeasureLightIntensity ( void  )

This function is responsible measure the change in the resistance of the sensor then we can make any calibration to convert this readings to light intensity readings.

Parameters
none
Returns
the value which indicates to the change in the sensor's resistance

this function is responsible for all the process of measuring the light intensity step by step:

  1. The sensor will read the physical value which is the light intensity
  2. The sensor is responsible for converting the physical value to an analog voltage
  3. The analog voltage will be entered to the adc channel
  4. The adc will convert the analog reading to digital reading
  5. Reading this digital value and convert it to analog value by multiplying it to the step of the adc
  6. Concert the analog reading to the physical value again and return it
Parameters
none
Returns
the value which indicates to the change in the sensor's resistance
