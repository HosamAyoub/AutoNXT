u32 HFSR_u32MeasureForce ( void  )

This function is responsible measure the change in the resistance of the sensor then we can make any calibration to convert this readings to Force readings.

Parameters
none
Returns
the value which indicates to the change in the sensor's resistance

this function is responsible for all the process of measuring the Force step by step:

  1. The sensor will read the physical value which is the Force
  2. The sensor is responsible for converting the physical value to an analog voltage
  3. The analog voltage will be entered to the adc channel
  4. The adc will convert the analog reading to digital reading
  5. Reading this digital value and convert it to analog value by multiplying it to the step of the adc
  6. Convert the analog reading to the physical value again and return it

the value which indicates to the change in the sensor's resistance
